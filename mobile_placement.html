Description

It is a Machine Learning project based upon the Placement Data set available on Kaggle. The original Notebooks can be viewed from here(EDA) and here(Model Building).

A detailed model search was conducted with various models such as Random Forest, Decision Tree classifier, KNN. after feature scaling, and one-hot encoding of the inputs. The Random Forest offered us an accuracy of 77% with the default parameters of the SK-learn library. To further enhance the accuracy, feature selection and hyperparameter tuning were performed on the data, which boosted the accuracy from 77% to 82.8%.

The model was then stored as a pickle file, and a simple GUI was made using Streamlit and hosted online where users can interact not only with the model but also the dataset. It is accessible using this Link.
